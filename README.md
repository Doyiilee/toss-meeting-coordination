# Align Time

Align Time은 회의 확정 가능 시간을 제안하는 일정 조율 서비스 MVP입니다.

## 목적

단순한 빈 시간 검색이 아니라, 참석자의 역할과 응답 여부를 분석해 회의가 확정될 수 있는 시간을 먼저 제안합니다.

## 실행 방법

`index.html`을 브라우저에서 열기

## 현재 구현 범위

- 초기 화면 (홈)
- 새 회의 만들기 / 회의 조건 입력 화면 (create.html)
- 참석자 조건 설정 화면 (participants.html)
- 확정 가능 시간 추천 화면 (results.html)
- 후보 시간 상세 / 확인 필요자 표시 화면 (detail.html)
- 회의 확정 완료 화면 (confirm.html)
